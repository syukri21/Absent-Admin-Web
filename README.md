# 👩‍💼 Absent Admin Web \(Front-End\)

## Installing

Install all dependencies is needed:

```
$ npm install
```

## Development

Run the application in development mode using this command

```bash
$ npm start
```

## Build

Build application using this command

```bash
$ npm run build
```
